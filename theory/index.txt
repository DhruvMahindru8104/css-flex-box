- one thing to remember jo flex box mein container hai uski height and width dene ki jrurat nhi because ismein container apni height and width items k hisab se dhal lega 
example - agr aapne container ki fixed height di to  container ki width items k hisab se apne aap adjust ho jayegi . and agr aapne container ki  fixed width di to  container ki height items k hisab se apne aap adjust ho jayegi  . but agr container ki width and height fixed hai to items flow ho jayengi mtlb items container k bahr nikl jayegi
